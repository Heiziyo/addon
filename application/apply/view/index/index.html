选择您要创建的应用类型

插件

为社区及 Discuz! 微社区提供各种增强功能的应用

  • 基准目录: /source/plugin/
  • 位于基准目录的子目录“插件标识/”下的所有文件
  • 只需提供简体 GBK 编码的安装 XML 文件，平台会自动生成其他编码的安装 XML 文件
模板

为社区提供各种模板界面的应用

  • 基准目录: /template/
  • 位于基准目录的子目录“模板标识/”下的所有文件
扩展

为社区提供高级扩展功能的应用

  • 基准目录: /
  • 位于基准目录下的所有扩展文件
  • 不能包含其他应用类型的基准目录
